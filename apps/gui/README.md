# Desktop GUI

```bash
uv run python grok_register_ttk.py
```

TTK app (`GrokRegisterGUI`):

- Form inputs: count, threads, proxy, mail provider, credentials paths
- Progress: status bar + scrolling log
- Actions: start / stop / clear / tutorial

Window title: **AI 注册机 · ai-register-machine**.  
Shares the Grok registration + CPA path with the CLI (not a demo shell).
